## Parameter Justification:

1. Embedding model: Use [`all-mpnet-base-v2`](https://sbert.net/docs/sentence_transformer/pretrained_models.html#original-models) which performs the best among all the models in Sentence Transformers. (https://huggingface.co/spaces/mteb/leaderboard)

2. UMAP Parameters: 
    - n_neighbors: higher -> global view, smaller -> local view (default_value: 15).
    - n_components: The dimensionality of the embeddings after reducing them (default: 5). Its effect is largest on the performance of HDBSCAN.
    - metric: cosine or euclidean. Used to compute the distances in high dimensional space.

3. HDBSCAN: 
    - min_cluster_size: The minimum size of a cluste (default: 10).
    - min_samples: Controls the number of outliers generated (deault: min_cluster_size). Setting this value significantly lower than min_cluster_size might help you reduce the amount of noise you will get.

4. CountVectorizer: We need to remove stop words and so we need to set stop_words='english'.

5. ClassTfidfTransformer: reduce_frequent_words=True.
